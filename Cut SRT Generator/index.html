How it works

  • Enter time ranges in simple text format
  • Select your video's frame rate
  • Timestamps align to valid frames
  • Download compliant SubRip (.srt) file

Technical details

Time ranges are parsed and normalized to seconds, then converted to frame-aligned timestamps using the selected FPS. Output files follow the standard SubRip specification.

Privacy

All processing is performed locally in your browser. No input data is uploaded, stored, or transmitted. Your content stays private.
